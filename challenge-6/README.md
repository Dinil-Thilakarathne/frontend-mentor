### What I learned

- how to select HTML element using nth:child method 

```CSS
.box:nth-child(1){
    background-color: var(--box1Color);
    border-radius: 10px 0 0 10px;
}
.box:nth-child(2){
    background-color: var(--box2Color);
}
.box:nth-child(3){
    background-color: var(--box3Color);
    border-radius: 0 10px 10px 0;
}
```
